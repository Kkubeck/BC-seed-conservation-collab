## Working Recommendation

For the fastest path to useful BC-native protocol pages:

1. Start with `Native Plant Network`.
2. Add `Pacific Northwest Plant Propagation Protocols` as a regional priority source.
3. Add `BC Tree Seed Centre` for trees.
4. Use `USDA PLANTS / Plant Guides` as structured-backfill-lite.
5. Use `Kew SID` and `Semantic Scholar` for gap filling and normalization, not first-pass ingestion.
